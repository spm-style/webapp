SPM CLI is powered by node.js and is only avaiable in npm registry at the moment.

Installing node.js and npm

Whether you're using OS X or Windows, to enjoy spm, you will need node.js and npm installed. The LTS version of node.js already includes npm. If you're using Linux, please check NodeSource's binary distributions.
