What is spm ?

spm is a style project manager. It was created in 2017 by Hervé Ehrhart and Adrien Guémy as an open source project to help developers and designers share modules (aka packages) of code in a very easy and efficient way.

The spm project contains two main parts:

  • The spm client, which is bundled with the Node.js platform, and is used as a command line tool to install and publish packages. It is simply called "spm". Its code is available as an open-source project: https://github.com/spm-style/cli
  • The spm registry service. When a user of the spm client runs the command spm install example-package, the spm client retrieves the package and all related dependencies from the spm registry service. The example output looks like this: user1-MacBook-Pro-2:~ user1$ spm install example-package spm http GET https://api.{{ domaine }}/example-package spm http 200 https://api.{{ domaine }}/example-package example-package@2.4.1 node_modules/example-package The registry is run as a free public service for anyone wanting to publish an open source package and for anyone to install an open source package.

spm is an association co-founded by spm's creator Hervé Ehrhart and Adrien Guémy.
